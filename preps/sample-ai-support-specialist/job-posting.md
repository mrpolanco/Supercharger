# Sample Posting: AI Platform Support Specialist

NimbusAI is hiring an AI Platform Support Specialist to help enterprise
customers troubleshoot API integrations, investigate production incidents, and
turn repeated support patterns into better docs and internal playbooks.

## Responsibilities

- Triage customer tickets involving API requests, authentication failures,
  rate limits, malformed JSON payloads, and unexpected model responses.
- Reproduce customer issues using command-line tools and small scripts.
- Query support data and operational logs to identify patterns across tickets.
- Partner with engineering on escalations by providing clear reproduction
  steps, impact, expected behavior, actual behavior, and relevant evidence.
- Write customer-facing replies that are technically accurate, kind, and clear.
- Improve support runbooks, troubleshooting checklists, and onboarding
  materials as the product changes.

## Required Qualifications

- 2+ years in technical support, solutions engineering, DevRel, SRE, or a
  similar technical customer-facing role.
- Comfortable reading API documentation and testing HTTP requests.
- Basic SQL for investigating customer records, event tables, or logs.
- Familiarity with JSON, authentication headers, and common HTTP status codes.
- Strong written communication for customers and internal engineering teams.
- Able to operate calmly during ambiguous incidents.

## Preferred Qualifications

- Experience supporting AI, developer tools, or technical SaaS products.
- Familiarity with context windows, token usage, rate limits, and streaming API
  responses.
- Basic Bash or scripting experience.
- Exposure to SSO, OAuth, SAML, or enterprise networking concepts.
